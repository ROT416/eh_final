< 기말과제 내용 정리 >


1. 코드 정리
라우팅 방식 변경 및 일부 코드를 파일로 분할. 


2. 데이터베이스 연동
HTML을 담는 상위 페이지 테이블, 하위 페이지테이블
유저 정보를 담는 유저 테이블, 덧글 정보를 담는 리플 테이블 


3. 회원가입 기능
회원 정보 입력 후 데이터베이스에 유저 정보 추가

데이터 베이스 내에 존재하는 아이디나 닉네임 값과 중복될 경우
또, 내용에 공백 또는 규칙에 맞지 않는 값을 입력한 경우에는
유저 정보를 추가하지 않음


4. 세션을 이용한 로그인 기능
데이터베이스 내의 유저 정보를 읽어와
아이디와 비번이 같을 경우 로그인되도록 구현

로그인 성공시 세션에 로그인 상태 정보와 닉네임 값을 저장하여
페이지 이동시 로그인 상태를 유지하고
로그인이나 회원가입으로의 접근을 막으며 메뉴 UI 변경
 

5. 덧글 기능 
데이터베이스의 join을 이용한 덧글 리스트 출력

세션을 활용해 로그인한 사람만 덧글 입력이 가능하게 하고
작성자만이 덧글을 삭제할 수 있도록 함
